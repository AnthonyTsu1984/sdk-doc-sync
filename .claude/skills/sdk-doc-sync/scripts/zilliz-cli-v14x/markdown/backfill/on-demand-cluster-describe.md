This operation shows details of one on-demand cluster.

## Usage

```bash
zilliz on-demand-cluster describe [OPTIONS]
```

## Options

- **--cluster-id** (*string*) -
  **[REQUIRED]**
  Specifies the target on-demand cluster ID.

## Example

```bash
zilliz on-demand-cluster describe --cluster-id clus_on_demand_xxx
```
